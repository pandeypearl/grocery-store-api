Grocery Api

This is a mock grocery items API intended especially for front-end develeopers like myself who would need an item list for simple projects. There is no API key required to interact with this API. You can interact with this data using GET, POST, PUT, PATCH, and DELETE. Please note that there is no database infrastructure for this API, therefore the use of PUT, PATCH, and DELETE will not be saved on the api server.

There are 5 sections :

1. Bakery - with different types of bagel, bread, cake, muffin, and pastry items.

As an example, this data can be accessed like this:

fetch('https://grocery-api-mexb.onrender.com/Bakery')
